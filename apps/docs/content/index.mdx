# Welcome to Multitenant Shell

A comprehensive multitenant application platform with enterprise-grade features, designed for modern SaaS applications.

## Key Features

- **Multi-tenant Architecture**: Isolated tenant data and configurations
- **User Management**: Complete user lifecycle management with role-based access
- **Authentication & Authorization**: JWT-based secure authentication
- **Role-based Access Control**: Fine-grained permissions system
- **REST API**: Complete API for all operations with interactive documentation

## Getting Started

Check out our [Quick Start](/docs/getting-started) guide to get up and running quickly.

## API Documentation

Explore our comprehensive [Users API](/docs/api/users) documentation with interactive examples and try-it-out functionality.

## Architecture

The multitenant shell is built with:

- **Backend**: NestJS with PostgreSQL and Redis
- **Frontend**: Next.js with TypeScript
- **Documentation**: Fumadocs with OpenAPI integration
- **Authentication**: JWT with refresh tokens
- **Database**: Multi-tenant PostgreSQL with Prisma ORM
