# API Reference

The My Application API provides programmatic access to all features.

## Base URL

```
https://api.myapp.com/v1
```

## Authentication

JWT Bearer token authentication

### Usage

```bash
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://api.myapp.com/v1/endpoint
```

## Quick Examples

### Get Items

```bash
curl -X GET "https://api.myapp.com/v1/items" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### Create Item

```bash
curl -X POST "https://api.myapp.com/v1/items" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "New Item", "description": "Item description"}'
```
