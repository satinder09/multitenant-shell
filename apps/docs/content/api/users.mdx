# Users API


## Get Users

`GET /users`

Retrieve all users

### Authentication Required

This endpoint requires a valid JWT token.

### Example Request

```bash
curl -X GET "https://api.myapp.com/v1/users" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"
```

### Response

```json
{
  "status": "success",
  "data": {}
}
```
