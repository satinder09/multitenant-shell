# Core API


## Get Items

`GET /items`

Retrieve all items

### Authentication Required

This endpoint requires a valid JWT token.

### Example Request

```bash
curl -X GET "https://api.myapp.com/v1/items" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"
```

### Response

```json
{
  "status": "success",
  "data": {}
}
```
