# Typography & UI Showcase

Welcome to the enhanced typography and UI experience! This page demonstrates the improved visual design with **Inter font**, smooth transitions, and better spacing.

## Enhanced Typography Features

### Beautiful Headings
The headings now use **Inter font** with optimized letter spacing and line heights for better readability.

### Improved Body Text
The body text features enhanced line spacing (1.75) and subtle letter spacing (-0.011em) for optimal reading comfort. The Inter font provides excellent legibility across all screen sizes.

### Code Blocks
```typescript
// Enhanced code blocks with JetBrains Mono font
interface EnhancedTypography {
  fontFamily: 'Inter' | 'JetBrains Mono';
  letterSpacing: string;
  lineHeight: number;
  fontWeight: 300 | 400 | 500 | 600 | 700 | 800;
}

const typography: EnhancedTypography = {
  fontFamily: 'Inter',
  letterSpacing: '-0.011em',
  lineHeight: 1.75,
  fontWeight: 400
};
```

## Interactive Elements

### Smooth Transitions
All interactive elements now feature smooth transitions with cubic-bezier easing functions:

- **Fast transitions**: 150ms for quick feedback
- **Normal transitions**: 250ms for balanced interactions  
- **Slow transitions**: 350ms for dramatic effects
